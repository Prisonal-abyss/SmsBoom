# 项目介绍
    本项目是针对Sms短信轰炸的一个Python3的实现方案。
    娱乐使用，共同学习，如果用于非法用途，后果自负.
 
# 环境依赖
 python 3.8
 requests == 2.20.0
 
# 目录结构描述
    ├── README.md           // 帮助文档
    
    ├── SmsBoom.py          // 项目主文件
    
 
# 使用说明
 1.使用“终端”定位到SmsBoom.py的所在目录;
 2.在“终端”中输入 python -m SmsBoom;
 3.尽情使用。
